# Mutual Fund Analytics Platform

## Project Overview

This project is an end-to-end Mutual Fund Analytics Platform built as part of the Bluestock Fintech Individual Capstone Project.

## Objectives

* Ingest mutual fund datasets and live NAV data
* Clean and validate data using Python and Pandas
* Load data into SQLite using SQLAlchemy
* Perform SQL-based analysis
* Build an interactive analytics dashboard

## Tech Stack

* Python
* Pandas
* NumPy
* SQLAlchemy
* SQLite
* Plotly
* Matplotlib
* Seaborn
* Jupyter Notebook
